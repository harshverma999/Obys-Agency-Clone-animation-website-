# Obys Agency Clone

A fully responsive and animated Obys Agency website clone built using HTML, CSS, and JavaScript.

🧠 About The Project

This is a frontend clone of the Obys Agency website created for learning and practice purposes.  
The project focuses on modern UI/UX design, smooth scrolling effects, and advanced animations.

---

✨ Features

- Smooth scrolling using Locomotive Scroll
- GSAP animations and transitions
- Custom cursor effects
- Interactive video section
- Project showcase with hover animations
- Responsive design

---

🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- GSAP
- ScrollTrigger
- Locomotive Scroll
- Shery.js

---

📂 Project Structure

├── index.html
├── style.css
├── script.js
├── assets / images / fonts

---

🎯 Purpose

This project was built to improve frontend animation skills and create a strong portfolio project.
